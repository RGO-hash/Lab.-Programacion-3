Listado de 102 Etiquetas HTML — Ejemplos y notas

Tabla completa con cada etiqueta, descripción, ejemplo y comentario adicional. (Explicaciones dentro del código.)

Nota: los ejemplos en la columna "Ejemplo de uso" muestran la sintaxis tal cual; están mostrados dentro de <code> para evitar que el navegador los ejecute.
Nº Etiqueta Descripción Ejemplo de uso Elemento adicional
1<a>Define un hipervínculo; enlaza a otra URL o recurso.<a href="https://example.com">Ir a Ejemplo</a>Atributo útil: target="_blank" para abrir en nueva pestaña.
2<p>Define un párrafo de texto.<p>Este es un párrafo.</p>Puede contener elementos en línea como <a>, <em>.
3<img>Inserta una imagen; auto-cerrada (void).<img src="foto.jpg" alt="Descripción">Siempre usar alt por accesibilidad; controlar tamaño con CSS.
4<div>Contenedor genérico en bloque para agrupar elementos.<div class="card">Contenido</div>Se usa para layout y aplicar estilos CSS.
5<h1>Encabezado de nivel 1 (más importante en la jerarquía).<h1>Título principal</h1>Solo uno por página para SEO y accesibilidad.
6<h2>Encabezado de nivel 2.<h2>Subtítulo</h2>Usado para secciones principales bajo <h1>.
7<h3>Encabezado de nivel 3 para subsecciones.<h3>Encabezado 3</h3>Mantén la jerarquía semántica.
8<h4>Encabezado de nivel 4.<h4>Encabezado 4</h4>Usado en estructuras complejas.
9<h5>Encabezado de nivel 5.<h5>Encabezado 5</h5>Raramente necesario.
10<h6>Encabezado de nivel 6 (menos importante).<h6>Encabezado 6</h6>Último nivel de jerarquía.
11<span>Contenedor en línea para aplicar estilo a parte del texto.<span class="rojo">Texto</span>Útil para estilizar palabras dentro de un párrafo.
12<strong>Texto con importancia semántica (aparece en negrita).<strong>Importante</strong>Mejor que <b> para semántica.
13<em>Texto enfatizado (aparece en cursiva).<em>Énfasis</em>Equivalente semántico a <i> cuando hay énfasis.
14<br>Inserta un salto de línea (void element).Linea1<br>Linea2No abusar dentro de bloques de texto, usar párrafos.
15<hr>Línea horizontal para separar contenidos.<hr>Útil para divisiones temáticas.
16<ul>Lista desordenada (viñetas).<ul><li>Uno</li></ul>Combinada con <li>.
17<ol>Lista ordenada (numerada).<ol><li>Primero</li></ol>Tiene atributos como start, type.
18<li>Elemento de una lista (ul/ol).<li>Elemento</li>Puede contener bloques y en línea.
19<table>Crea una tabla para datos tabulares.<table><tr><td>Dato</td></tr></table>Usar para datos tabulares, no para layout.
20<tr>Fila de una tabla.<tr><td>Celda</td></tr>Contiene celdas (<td> o <th>).
21<td>Celda de datos en una fila de tabla.<td>Contenido</td>Por defecto alineada a la izquierda.
22<th>Celda de encabezado de tabla (normalmente en negrita/centrado).<th>Título</th>Usar en la primera fila o en <thead>.
23<thead>Agrupa la sección de encabezados de la tabla.<thead><tr><th>Col</th></tr></thead>Ayuda en accesibilidad y estilo.
24<tbody>Agrupa el cuerpo principal de la tabla.<tbody><tr><td>Dato</td></tr></tbody>Separar <thead> de <tfoot> es buena práctica.
25<tfoot>Agrupa el pie de tabla (totales, resúmenes).<tfoot><tr><td>Total</td></tr></tfoot>Puede colocarse antes o después de tbody; navegador lo ajusta.
26<form>Contenedor para campos que envían datos a un servidor.<form action="procesar.php" method="post">...</form>Atributos: action, method.
27<input>Campo de entrada (muchos tipos disponibles con type).<input type="text" name="usuario">Tipos comunes: text, email, password, number.
28<label>Etiqueta asociada a un control de formulario para accesibilidad.<label for="nom">Nombre:</label>Usar for enlazando al id del control.
29<textarea>Área de texto multilinea para comentarios largos.<textarea rows="4">Escribe...</textarea>Soporta atributos: rows, cols, maxlength.
30<button>Botón interactivo (submit, reset, button).<button type="submit">Enviar</button>Preferir <button> sobre <input type="submit"> para mayor flexibilidad.
31<select>Lista desplegable para escoger una opción.<select><option>A</option></select>Combinar con <option> y <optgroup> para agrupar.
32<option>Elemento de opción dentro de <select>.<option value="1">Uno</option>Atributo selected marca la opción por defecto.
33<optgroup>Agrupa opciones dentro de un <select>.<optgroup label="Grupo 1">...</optgroup>Mejora usabilidad en listas largas.
34<fieldset>Agrupa campos del formulario visualmente.<fieldset><legend>Datos</legend>...</fieldset>Útil para accesibilidad y organización.
35<legend>Título para un <fieldset>.<legend>Información personal</legend>Debe estar dentro de <fieldset>.
36<iframe>Inserta otra página o recurso embebido dentro del documento.<iframe src="mapa.html"></iframe>Atributos útiles: width, height, title (accesibilidad).
37<audio>Reproductor de audio nativo.<audio controls src="cancion.mp3"></audio>Atributos: controls, autoplay, loop, preload.
38<video>Reproductor de vídeo nativo.<video controls><source src="v.mp4"></video>Soporta subtítulos con <track>.
39<source>Define una fuente alternativa para audio/video.<source src="v.mp4" type="video/mp4">Colocar varias para compatibilidad de formato.
40<track>Subtítulos o pistas para multimedia.<track src="sub.vtt" kind="subtitles" srclang="es">Mejora accesibilidad del contenido audiovisual.
41<canvas>Área para dibujar gráficos dinámicos mediante JavaScript.<canvas id="c"></canvas>Necesita script para dibujar en 2D/3D.
42<svg>Contenedor para gráficos vectoriales SVG (escalables).<svg width="100" height="100"><circle cx="50" cy="50" r="40"/></svg>Escalable sin pérdida; se puede estilizar con CSS.
43<path>Elemento SVG para dibujar rutas complejas (curvas, líneas).<path d="M150 0 L75 200 L225 200 Z"/>Parte de SVG; define la forma con 'd'.
44<link>Vincula recursos externos, como hojas de estilo (CSS) o iconos.<link rel="stylesheet" href="styles.css">Se coloca dentro de <head>.
45<style>Define estilos CSS en línea dentro del documento.<style>body{background:red;}</style>Mejor usar archivos externos para mantenimiento.
46<script>Inserta o referencia código JavaScript.<script src="app.js"></script>Si se coloca en <head>, usar defer o async para rendimiento.
47<noscript>Contenido alternativo mostrado cuando JavaScript está deshabilitado.<noscript>Activa JS para ver esta página</noscript>Útil para accesibilidad y SEO.
48<meta>Metadatos del documento (charset, description, viewport...).<meta name="description" content="... ">Se coloca en <head>.
49<title>Título que aparece en la pestaña del navegador; importante para SEO.<title>Mi página</title>Visible en resultados de búsqueda.
50<base>Define la URL base para enlaces relativos.<base href="https://example.com/">Afecta a todos los enlaces relativos; usar con cuidado.
51<header>Sección de encabezado de la página o de una sección.<header><h1>Bienvenido</h1></header>Suele incluir logo y navegación.
52<nav>Área que agrupa enlaces de navegación principales.<nav><a href='#'>Inicio</a></nav>Ayuda a usuarios y a motores de búsqueda.
53<main>Contenedor del contenido principal de la página (solo uno por documento).<main>Contenido principal</main>Importante para accesibilidad (skip links).
54<section>Sección temática dentro del contenido.<section><h2>Noticias</h2></section>Útil para dividir contenido por temas.
55<article>Contenido independiente que puede distribuirse por separado (post, noticia).<article><h2>Post</h2></article>Ideal para entradas de blog.
56<aside>Contenido complementario (barra lateral, notas, anuncios).<aside>Enlaces relacionados</aside>No es el contenido principal.
57<footer>Pie de página con información de contacto y créditos.<footer>&copy; 2025</footer>Frecuentemente contiene <address>.
58<address>Información de contacto (dirección, email).<address>contacto@ejemplo.com</address>Usar dentro de <footer> o similar.
59<figure>Contenedor para medios (imagen, gráfico) con leyenda.<figure><img src='f.jpg'><figcaption>Leyenda</figcaption></figure>Usar con <figcaption> para describir la imagen.
60<figcaption>Leyenda o pie para la figura.<figcaption>Descripción</figcaption>Debe estar dentro de <figure>.
61<mark>Resalta texto (por ejemplo búsqueda).<mark>Resultado</mark>Color por defecto: amarillo.
62<time>Representa fecha y/o hora en formato legible y máquina.<time datetime="2025-09-15">15 Sep 2025</time>Usar atributo datetime para datos legibles por máquinas.
63<abbr>Abreviatura o sigla con su expansión en title.<abbr title="World Health Organization">WHO</abbr>Muestra el significado al pasar el cursor.
64<code>Fragmento de código en línea (monoespaciado).<code>console.log('Hola')</code>Ideal para ejemplos de programación.
65<pre>Texto preformateado (mantiene espacios y saltos).<pre> indented text</pre>Útil para mostrar código con formato.
66<kbd>Indica entrada de teclado (teclas o atajos).<kbd>Ctrl</kbd> + <kbd>C</kbd>Usado en documentación de atajos.
67<samp>Salida de un programa o comando (ej. consola).<samp>Error: 404</samp>Semántica para salidas de sistemas.
68<var>Representa una variable (en matemáticas/código).<var>x</var> = 10Monoespaciado por defecto en algunos navegadores.
69<small>Texto de tamaño reducido; notas o disclaimers.<small>Nota al pie</small>No usar para contenido importante.
70<sup>Superíndice (p. ej. potencias).x<sup>2</sup>Útil en fórmulas matemáticas.
71<sub>Subíndice (p. ej. fórmulas químicas).H<sub>2</sub>OUsado en notación científica/química.
72<b>Texto en negrita sin significado semántico adicional.<b>Negrita</b>Preferir <strong> para indicar importancia.
73<i>Texto en cursiva sin énfasis semántico.<i>Cursiva</i>Preferir <em> para énfasis.
74<u>Subraya texto.<u>Subrayado</u>Evitar confundir con enlaces; usar con moderación.
75<del>Indica texto eliminado (tachado).<del>Texto viejo</del>Útil para mostrar ediciones o cambios.
76<ins>Texto insertado (opuesto de <del>).<ins>Nuevo texto</ins>Puede estilizarse para mostrar actualizaciones.
77<q>Cita corta en línea (automáticamente añade comillas en algunos navegadores).<q>El saber es poder</q>Para citas breves en texto corriente.
78<blockquote>Cita en bloque para textos más largos.<blockquote>Texto citado...</blockquote>Suele renderizarse con sangría.
79<cite>Referencia a una obra (libro, artículo, etc.).<cite>Don Quijote</cite>Normalmente en cursiva.
80<bdi>Aíslax texto con dirección independiente (bidireccionalidad).<bdi>العربية</bdi>Útil al mezclar idiomas con direcciones distintas.
81<bdo>Sobrescribe la dirección del texto (dir='rtl'|'ltr').<bdo dir="rtl">Texto</bdo>Forzar orden de lectura cuando sea necesario.
82<wbr>Punto opcional de quiebre de palabra (word break).hiper<wbr>textoNo visible, ayuda en palabras largas.
83<details>Contenedor desplegable; se puede abrir/colapsar.<details><summary>Más info</summary><p>Contenido oculto</p></details>Útil para FAQ y contenido colapsable.
84<summary>Encabezado clicable de <details>.<summary>Haz clic</summary>Debe estar directamente dentro de <details>.
85<dialog>Cuadro de diálogo/modal nativo del navegador.<dialog open>Hola</dialog>Funcionalidad limitada en algunos navegadores; se usa con JS para control.
86<menu>Lista de comandos o menú; su uso ha variado con el tiempo.<menu><li>Opción</li></menu>En HTML5 su uso es menos común; reemplazado por <ul> habitualmente.
87<menuitem>Elemento de menú contextual (obsoleto en HTML5).<menuitem label="Guardar"></menuitem>No recomendable: soporte limitado/obsoleto.
88<progress>Barra de progreso que muestra avance de una tarea.<progress value="70" max="100"></progress>Ideal para progresos de carga.
89<meter>Representa un valor dentro de un rango (ej. nivel, ratio).<meter value="0.6">60%</meter>Diferente semánticamente de <progress>.
90<output>Elemento para mostrar el resultado de cálculos en formularios.<output name="res">42</output>Puede usarse con JS o con atributos form-associated.
91<template>Contenedor de contenido no renderizado hasta que se instancia con JS.<template><div>Plantilla</div></template>Perfecto para clones dinámicos con JS.
92<slot>Punto de inserción en Web Components (Shadow DOM).<slot>Contenido por defecto</slot>Solo en componentes web; no para HTML clásico.
93<picture>Contenedor para imágenes adaptables (responsive, formatos).<picture><source srcset="img.webp" type="image/webp"><img src="img.jpg" alt="X"></picture>Permite servir WebP, AVIF, etc. según soporte del navegador.
94<object>Inserta recursos externos (PDF, applet, etc.).<object data="archivo.pdf" type="application/pdf"></object>Puede usar <param> para opciones.
95<embed>Inserta contenido externo embebible.<embed src="anim.swf">Uso reducido modernamente (plugins obsoletos).
96<param>Define parámetros para <object>.<param name="autoplay" value="true">Acompaña a <object> para pasar opciones.
97<col>Define propiedades de una columna en tablas.<col span="2">Se usa dentro de <colgroup> para estilo.
98<colgroup>Agrupa columnas en una tabla para aplicar estilos.<colgroup><col></colgroup>Útil para diseño de tablas complejas.
99<dl>Lista de definiciones (término + definición).<dl><dt>Término</dt><dd>Definición</dd></dl>Semántica adecuada para glosarios.
100<dt>Término en una lista de definiciones (<dl>).<dt>HTML</dt>Siempre en contexto de <dl>.
101<dd>Definición de un término (<dt>).<dd>Lenguaje de marcado</dd>Complementa <dt> en <dl>.
102<html>Elemento raíz del documento HTML; contiene <head> y <body>.<html lang="es">...</html>Atributo lang mejora accesibilidad y SEO.
