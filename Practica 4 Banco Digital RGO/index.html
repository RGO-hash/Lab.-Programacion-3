🎓 Proyecto Académico - Banco RGO

Este proyecto representa la culminación de conocimientos adquiridos en la Licenciatura en Informática, demostrando competencias en desarrollo web, diseño de interfaces y programación funcional.

📋 Objetivos del Proyecto

  • Demostrar dominio de tecnologías web fundamentales (HTML5, CSS3, JavaScript)
  • Implementar arquitectura de navegación inspirada en sitios bancarios profesionales
  • Crear una experiencia de usuario intuitiva y moderna
  • Desarrollar contenido original y estructurado para servicios financieros
  • Aplicar principios de responsive design para múltiples dispositivos
  • Integrar animaciones y efectos visuales profesionales

🎯 Metodología de Desarrollo

1️⃣ Análisis e Investigación

Estudio de la arquitectura web de Banreservas como referencia, análisis de mejores prácticas en UX bancario.

2️⃣ Diseño y Planificación

Creación de wireframes, definición de paleta de colores, estructura de navegación y contenido.

3️⃣ Desarrollo Frontend

Implementación con HTML semántico, CSS moderno con Flexbox/Grid, JavaScript funcional.

4️⃣ Testing y Optimización

Pruebas de responsividad, optimización de rendimiento, validación de código.
